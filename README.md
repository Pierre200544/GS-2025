GS-2025 - Projeto FIAP (React + Tailwind) - Completo

Como usar (instalação):
1. Tenha Node.js instalado.
2. No terminal, na pasta do projeto, rode:
   npm install
   npm run dev
3. A aplicação usa Vite; o dev server geralmente abre em http://localhost:5173

O que está incluído:
- Projeto React + Vite + Tailwind pronto
- src/pages/Home.jsx -> listagem e lógica principal
- src/components -> Card, Modal, SearchBar
- src/data/profiles.json -> 60 perfis simulados
- src/images -> 60 imagens placeholder geradas
- tailwind.config.js / postcss.config.cjs configurados
- init_git.sh -> script para inicializar repo e criar 10 commits de demonstração

Script Git (execute localmente):
  chmod +x init_git.sh
  ./init_git.sh

Observação: instale dependências localmente; aqui foi criado o scaffold e todos os arquivos.
